# Contributing to Champion Calculations

Thanks for contributing.

This project is currently maintained by an individual (not a company). The project is open source under `AGPL-3.0-or-later`, with a Contributor License Agreement (`CLA.md`) so the maintainer can keep future licensing options open.

By opening a pull request, you confirm that:
- You have the right to submit the contribution.
- You agree to the Contributor License Agreement in `CLA.md`.
- You understand your contribution will be released under the repository license and may be relicensed by the maintainer as described in `CLA.md`.

## Contribution Flow

1. Fork the repository and create a focused branch.
2. Make your change with clear commit messages.
3. Run relevant local checks for the area you changed.
4. Open a pull request with a short problem/solution summary.
5. Add this exact line to the PR description:

`I have read and agree to CLA.md for this contribution.`

## Scope and Legal Notes

- Keep contributions scoped to this repository's goals.
- Do not submit third-party code or data unless you have clear rights to do so.
- Third-party game/IP assets and data remain under their own terms. See `THIRD_PARTY_NOTICES.md`.
